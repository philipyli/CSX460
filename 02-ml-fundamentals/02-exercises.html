Exercise 2:

Write a loss functions for calculating: - Root Mean Square Error - Mean Absolute Error - Median Absolute Error

All functions should accept two arguments:

rmse <- function(y,yhat) {
  ( y - yhat )^2  %>% mean %>% sqrt 
}

mae <- function(y, yhat) {
  ( y - yhat )  %>% abs %>% mean
}

medae <- function(y, yhat) { 
  ( y - yhat )  %>% abs %>% median
}

Use these functions to evaluate the loss/performance of: - the naive guess

rmse(dat$FE, naive_guess)

mae(dat$FE, naive_guess)
## [1] 6.319942
